# Project Technical Profile

*   **Technical Name**: **JuiceBox** (Real-Time Clipboard-Triggered NLP Jargon Classifier & Generative Career Narrative Rewrite Engine)
*   **Aesthetic Codename**: Project Papaya (PRJ-PAPAYA-2026)
*   **Description**: A low-latency background service and UI companion that intercepts native OS clipboard streams, performs local lexical analysis and grammatical tokenization using a spaCy-driven matcher to flag corporate jargon density (Corporate Robot Score), and dispatches structured payloads to the Gemini API for generative, narrative-driven career introductions.
